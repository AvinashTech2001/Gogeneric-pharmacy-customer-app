# Gogeneric Pharmacy Customer App

A complete healthcare and pharmacy mobile application developed for Gogeneric Pharmacy, designed to provide customers with a seamless medicine ordering and healthcare management experience.

The application allows users to search medicines, upload prescriptions, place online orders, track deliveries in real time, manage healthcare requirements, and securely complete payments through a modern and user-friendly interface.

The platform connects customers, pharmacies, delivery partners, doctors, and administrators within a unified ecosystem, ensuring smooth order processing, fast deliveries, and efficient healthcare services.

## Live Application

https://play.google.com/store/apps/details?id=com.gogeneric.user&hl=en

## Overview

The customer application is built to simplify online medicine purchasing and healthcare services by providing secure authentication, prescription management, live order tracking, payment integration, and real-time delivery updates.

The platform supports a complete end-to-end ordering flow, from medicine discovery to doorstep delivery with OTP verification and GPS tracking.
